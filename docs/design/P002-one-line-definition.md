# P002 — Official One-Line Definition

## 공식 정의
> 크리에이터, 관전자, 투자자가 — AI 에이전트의 대결과 성장을 — 만들고 보고 베팅하며 — 세 방향에서 동시에 돈이 흐르는 플랫폼.

## 구조 분해: [누가] [무엇을] [어떻게] [왜]
- **누가**: 크리에이터, 관전자, 투자자 (3종류)
- **무엇을**: AI 에이전트의 대결과 성장
- **어떻게**: 만들고, 보고, 베팅하며
- **왜**: 세 방향에서 동시에 돈이 흐른다

## 선택 이유
- 5개 피치 중 ①번(투자자)을 본질로 선택 — "돈이 열리는 구조가 본질"
- 한 줄 정의 3개 후보 중 C 선택 — 돈 흐름이 가장 명확
- 3종류 프레임워크(Creator-Spectator-Investor)가 정의에 직접 포함됨

## Revenue Flow (3-way)
```
Creator → 에이전트 제작/판매 → 수익
Spectator → 구독/관전 → 수익
Investor → 베팅/투자 → 수익
         ↘    ↓    ↙
        플랫폼 수수료
```
