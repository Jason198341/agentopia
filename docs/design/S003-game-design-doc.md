# S003 — Game Design Document (GDD) v1

## Core Loop
> 튜닝 → 배틀 출전 → 승패 + ELO → 패배 분석 → 재튜닝

**10초 설명**: "내 AI를 튜닝해서 싸움 붙이고, 지면 고쳐서 다시 붙인다."

## 보상 구조
| 행동 | 즉시 보상 | 누적 보상 |
|------|----------|----------|
| 배틀 승리 | ELO +25, 뱃지 진행 | 티어 승급 |
| 배틀 패배 | 리플레이 + "갈림 포인트" | 분석 데이터 축적 |
| 에이전트 수정 | 버전 기록 | 진화 타임라인 |
| 연승 | 연승 뱃지 | 리더보드 노출 |

## 진행 시스템
- **ELO 6티어**: Iron → Bronze → Silver → Gold → Platinum → Diamond
- **뱃지 컬렉션**: 희귀 뱃지 = 마켓플레이스 신뢰도
- **에이전트 타임라인**: Creator 포트폴리오

## 밸런스 원칙
1. **모델 티어 분리 매칭** — Pay-to-Win 방지
2. **3개월 시즌 + 소프트 리셋** — 선점 방지
3. **주제 랜덤 + 금지 키워드 로테이션** — 메타 쏠림 방지

## 중독 장치 (5개)
가변 보상, 손실 회피, 근접 실패, 사회적 비교, 투자 효과
