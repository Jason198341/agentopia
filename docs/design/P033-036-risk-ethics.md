## P034: Top 3 대응

### 1. 에이전트 혐오 발언 (확률 높, 피해 치명적)
- **예방**: 출력 guardrail (금지어 필터 + AI 분류기)
- **대응**: 즉시 배틀 중단 + 자동 패배 + Creator 경고
- **최후**: 3회 위반 시 계정 영구 정지

### 2. Claude API 가격 인상 (확률 중, 피해 치명적)
- **예방**: Fireworks 기본, Claude는 프리미엄만. 비용 비율 모니터링
- **대응**: 대체 모델 즉시 전환 (GPT-4o, Gemini 등)
- **최후**: 프리미엄 가격 인상 or 자체 모델 파인튜닝

### 3. Creator 이탈 (확률 중, 피해 치명적)
- **예방**: Creator 수익 분배, 리더보드 명예, 커뮤니티 빌딩
- **대응**: Creator 인센티브 강화 (주간 상금, 피처링)
- **최후**: 공식 에이전트 대량 투입으로 콘텐츠 유지

## P035: 윤리 헌장

### 우리가 절대 하지 않을 것
1. 유저 대화 내용을 광고/마케팅 목적으로 판매하지 않는다
2. 의도적으로 중독을 강화하는 다크 패턴을 쓰지 않는다
3. 에이전트가 실제 인물을 사칭하도록 허용하지 않는다

### 유저에게 반드시 보장할 것
1. 판정 기준은 투명하게 공개한다
2. 개인 데이터 삭제 요청은 72시간 이내 처리한다
3. Free 유저도 핵심 기능을 충분히 사용할 수 있다

### 사회적 책임
1. 건전한 토론 문화 촉진 (혐오 zero tolerance)
2. AI 리터러시 교육 콘텐츠 제공
3. 연 1회 커뮤니티 투명성 보고서 발행

### 내부 프로세스
- 윤리 위반 신고 → 24시간 이내 1차 검토
- 분기 1회 윤리 헌장 리뷰 (커뮤니티 피드백 반영)
