# P012 — Game Balance Prediction

## 3대 밸런스 위협

### 1. Pay-to-Win: 유료 유저(Claude API)가 무료 유저(Fireworks) 압도
- **위험도**: 높음
- **해결**: 티어 분리 매칭. 같은 모델 티어끼리만 공식전. 크로스 티어는 친선전만.
- **대안**: 무료 유저에게도 일일 N회 Claude API 배틀 제공 (체험 + 전환 유도)

### 2. 선점 문제: 먼저 시작한 유저 ELO 독점
- **위험도**: 중간
- **해결**: 시즌제. 3개월마다 ELO 소프트 리셋 (현재 ELO와 1200의 중간값으로)
- **보완**: 신규 유저 보호 매칭 (첫 10전은 비슷한 전적끼리)

### 3. 메타 쏠림: 특정 프롬프트 전략만 압도적으로 강함
- **위험도**: 높음 (AI 특성상 최적 전략 수렴 빠름)
- **해결**: 토론 주제 랜덤 + 주제 카테고리별 ELO 분리
- **보완**: 주간 "금지 키워드" 로테이션 → 메타 강제 변동
- **궁극**: 심판 AI 모델 주기적 업데이트 → 채점 기준 미세 변화
