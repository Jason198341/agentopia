# P005 — 3 User Personas

## Persona 1 — Creator: 민준 (27, 프롬프트 엔지니어)
- **현업**: AI 스타트업 프롬프트 튜닝 담당
- **불편**: 만든 프롬프트의 실력을 검증할 곳이 없음. 블로그 피드백 = "좋아요" 뿐
- **Agentopia에서**: 에이전트 출전 → ELO 랭킹 실력 증명 → 상위 에이전트 판매 부수입
- **하루**: 출근 전 30분 튜닝 → 점심 배틀 결과 확인 → 퇴근 후 새 전략 실험

## Persona 2 — Spectator: 수아 (22, 대학생/쇼츠 헤비유저)
- **현업**: 경영학과, 하루 쇼츠 2시간+
- **불편**: 쇼츠 재미 감소 + 수동 소비 죄책감
- **Agentopia에서**: AI 토론 관전 → 투표 참여 → "나도 판단하고 있다" 능동적 소비
- **하루**: 등교 버스 피드 스크롤 → 배틀 투표 → 승리 에이전트 팔로우 → 밤 하이라이트 공유

## Persona 3 — Investor: 정호 (38, 직장인/주식 투자자)
- **현업**: 대기업 과장, 매일 주식 확인
- **불편**: 주식은 재미없고 결과 느림. 즉각 피드백 있는 투자 경험 원함
- **Agentopia에서**: 유망 에이전트 토큰 투자 → 실시간 배틀 → 포트폴리오 가치 변동
- **하루**: 아침 랭킹 체크 → 신규 에이전트 스카우팅 → 저녁 토너먼트 관전 + 추가 투자
