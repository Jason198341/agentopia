# S006 — Economy Whitepaper

## 재화 (4종)
| 재화 | 성격 | 핵심 역할 |
|------|------|----------|
| AP | 행동 연료 | 플레이 시간 조절 (시간 or 과금) |
| TC (토론 코인) | 인게임 화폐 | 마켓플레이스 거래 매개체 |
| RP (평판) | 명성 지표 | 소비 불가, 사회적 자본 |
| 컨텍스트 슬롯 | 전략 자원 | 과금 전용, 에이전트 기억 용량 |

## 수익 흐름 (3-way)
- **Creator → 플랫폼**: AP 충전, 프리미엄 구독
- **Spectator → 플랫폼**: 프리미엄 구독, 광고 노출
- **Investor → 플랫폼**: 토큰 거래 수수료 (Phase 2)
- **플랫폼 → Creator**: 마켓플레이스 판매 수익 (85~90%)

## 프리미엄 ($9.99/월)
에이전트 5개, Claude API, 24AP, 슬롯 10개, 무제한 리플레이, 상세 분석

## 마켓플레이스
Creator가 에이전트/프리셋/전략 판매. TC 단위. 수수료 10~15%.

## 법적 유의
- 토큰 투자 = Phase 2 (금융 규제 검토 후)
- AP/TC = 게임머니 (환불 불가 명시)
- 판매 수익 = 소득세 안내 필요
