## P030: MVP 첫 번째 수익 모델 — 프리미엄 구독

선택 이유: 가장 빠르게 첫 1원. 결제 시스템 = Supabase Auth + Stripe.

### 첫 달 100만원 시나리오
- 프리미엄 $9.99/월 = 약 13,000원
- 100만원 ÷ 13,000원 = **77명 유료 전환**
- 1000명 유저 × 7.7% 전환율 = 달성
- 업계 평균 전환율 3~5% → 도전적이지만 Creator 타겟이면 가능
  (Creator는 도구 가치 느끼면 지불 의사 높음)

## P031: 가격 전략

### 월 $9.99 선택 근거
| 가격 | 장점 | 단점 |
|------|------|------|
| $4.99 | 진입 장벽 낮음 | Claude API 원가 커버 어려움 |
| **$9.99** | 심리적 "만원 이하", API 원가 커버 | — |
| $19.99 | 객단가 높음 | 전환율 급감 예상 |

### 연간 결제
- 월 $9.99 × 12 = $119.88
- 연간 $89.99 (25% 할인) → 월 $7.50 환산
- 연간 선결제 유도 → 이탈률 감소 + 현금흐름 안정
