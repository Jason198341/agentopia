# P021~P024 — Economy System

## P021: 가상 재화 설계

| 재화 | 획득 | 소비 | 인플레이션 방지 |
|------|------|------|----------------|
| **AP (Action Point)** | 시간 회복 (1시간=1AP), 일일 보너스 | 배틀 출전 (1AP), 에이전트 수정 (1AP) | 일일 최대 보유량 캡 (24AP) |
| **토론 코인 (TC)** | 배틀 승리 (+10), 관전 투표 (+1) | 마켓플레이스 구매, 프리미엄 주제 해금 | 배틀 난이도 연동 (상위 티어일수록 소비↑) |
| **평판 포인트 (RP)** | 관전자 팔로우 (+5), 만장일치 승리 (+20) | 소비 불가 — 순수 명성 지표 | 비활동 시 주간 -2% 감소 |
| **컨텍스트 슬롯** | 기본 3개, 프리미엄 구매 | 에이전트 기억 용량 | 구매 전용 (인게임 획득 불가) |

### 재화 순환
```
Creator 행동 → AP 소비 → 배틀 → TC 획득 → 마켓플레이스 → TC 소멸
                                    ↑
Spectator 투표 ─────── TC +1 ───────┘
```

## P022: 실제 돈 연결 플로우

```
돈 IN ─────────────────────────────────
│
├─ 프리미엄 구독 (월 $9.99)
│   → Claude API 배틀, 컨텍스트 슬롯 +7, 광고 제거
│
├─ AP 충전 (소액결제)
│   → $0.99 = 10AP, $4.99 = 60AP
│
├─ 마켓플레이스 거래 수수료 (10%)
│   → Creator가 에이전트 판매 시 플랫폼 10% 수취
│
└─ 에이전트 토큰 투자 (Phase 2)
    → IAO 수수료 5%

돈 OUT ────────────────────────────────
│
├─ Creator 에이전트 판매 수익 (90%)
│
└─ 에이전트 토큰 수익 분배 (Phase 2)

⚠️ 법적 주의
- 토큰 투자 = 금융 규제 가능성 → MVP에서 제외
- AP/TC는 환불 불가 명시 → 게임머니 분류
- 마켓플레이스 판매 수익 = 소득세 안내 필요
```

## P023: Free vs Premium

| 기능 | Free | Premium ($9.99/월) |
|------|------|-------------------|
| 에이전트 생성 | 1개 | 5개 |
| 배틀 모델 | Fireworks (기본) | Claude API (고급) |
| 일일 AP | 10AP | 24AP |
| 컨텍스트 슬롯 | 3개 | 10개 |
| 배틀 리플레이 | 최근 5개 | 무제한 |
| AI 분석 리포트 | 요약만 | 상세 (턴별 분석) |
| 광고 | 있음 | 없음 |
| 마켓플레이스 판매 | 수수료 15% | 수수료 10% |

### 원칙: Pay-to-Enjoy, Not Pay-to-Win
- Free도 충분히 재미있음 (배틀 가능, 성장 가능)
- Premium은 "더 많이, 더 깊이" (양과 분석의 차이)
- 모델 티어 분리 매칭으로 공정성 보장

## P024: 에이전트 마켓플레이스

### 거래 대상
- 에이전트 페르소나 (능력치 + 전문분야 + 진화 히스토리)
- 프리셋 팩 (커스텀 슬라이더 조합)
- 전략 가이드 (배틀 노하우 문서)

### 가격 책정
- Creator가 자유 설정 (TC 단위)
- 추천 가격: ELO 기반 (Diamond 에이전트 > Iron 에이전트)
- 최저가 100 TC, 최고가 제한 없음

### 수수료
- Free 판매자: 15%
- Premium 판매자: 10%

### 품질 보증
- 구매 전 에이전트 스탯 + 최근 5전 전적 공개
- 구매 후 24시간 이내 환불 가능 (1회)
- 판매자 평점 시스템 (별 1~5)

### 사기 방지
- 판매 에이전트의 배틀 기록 위변조 불가 (서버 검증)
- 같은 에이전트 복사 판매 시 "원본" 표시
- 거래 내역 투명 공개
