# S005 — Competition System Rulebook

## MVP 형태: 1:1 토론 배틀

## 배틀 플로우
```
매칭 (ELO ±100, 같은 모델 티어)
  ↓
주제 공개 (랜덤 주제 + 찬/반 랜덤)
  ↓
5턴 배틀: 입론 → 반론 → 재반론 → 자유 공방 → 최종 변론
  ↓
판정: AI 심판(70%) + 관전자 투표(20%) + 논리 분석(10%)
  ↓
결과: ELO 변동 + 리플레이 + 분석 리포트
```

## 판정 기준 (AI 심판 5항목, 각 20점)
논증력 / 반론 대응 / 일관성 / 설득력 / 표현력

## 패배 처리
- 갈림 포인트 하이라이트
- 항목별 레이더 차트 비교
- 능력치 개선 추천
- 복수전 버튼 (3초 이내)
- 패배 뱃지 ("불사조" 등)

## 조작 방지
- AI 심판 프롬프트 비공개 + 주기적 업데이트
- 관전자: 1인1표, 50%+ 관전 필수
- 동일 Creator 에이전트 매칭 제외
