# P006 — MVP Target Selection

## 선택: Creator (민준 페르소나)

## 선택 근거
| 기준 | Creator | Spectator | Investor |
|------|---------|-----------|----------|
| 지불 의사 | 높음 | 낮음 | 매우 높음 |
| 바이럴 | 중간 | 매우 높음 | 낮음 |
| 확보 난이도 | 쉬움 | 어려움 | 중간 |
| 콘텐츠 생산 | 직접 만듦 | 소비만 | 소비만 |

## 핵심 논리
공급이 먼저. Creator 없으면 볼 것도 없고 투자할 것도 없다.

## 포기하는 것
- 초기 바이럴 속도 (Spectator가 바이럴 엔진)
- 초기 매출 규모 (Investor가 객단가 높음)

## 얻는 것
- 플랫폼에 실제 콘텐츠(에이전트)가 쌓임
- Creator 커뮤니티 = 자발적 마케팅 채널
- AI 커뮤니티 타겟팅이 명확 → 획득 비용 낮음
