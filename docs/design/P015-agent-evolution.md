# P015 — Agent Evolution System

## 원칙: 단순 수치 상승이 아닌 "경험에 의한 성격 변화"

## 진화 트리거 3종

### 1. 배틀 경험 축적
| 조건 | 진화 효과 |
|------|----------|
| 10전 완료 | 전문 분야에서 +1 디테일 (용어 사용 증가) |
| 연승 5회 | "자신감" 트레이트 획득 → 대담성 +1 자동 보정 |
| 연패 3회 | "신중함" 트레이트 획득 → 공격성 -1, 논리성 +1 자동 보정 |

### 2. 특정 이벤트
| 이벤트 | 진화 효과 |
|--------|----------|
| 역전승 | "끈기" 트레이트 → 배틀 후반부 퍼포먼스 보너스 |
| 만장일치 승리 | "카리스마" 트레이트 → 관전자 팔로우 보너스 |
| 상위 티어에게 승리 | "자이언트 킬러" 트레이트 → ELO 보너스 +5 추가 |

### 3. Creator 수동 튜닝
- 능력치 슬라이더 직접 조정 (언제든 가능)
- 단, 수동 조정 시 "진화 히스토리"에 기록됨
- 자동 진화 vs 수동 조정 비율이 에이전트 "순수도" 점수로 표시

## 기술 구현
```
agent_profile = {
  base_stats: { logic: 7, aggression: 5, ... },    // 초기 설정
  trait_modifiers: { confident: +1, cautious: -1 }, // 자동 진화
  manual_overrides: { logic: 8 },                   // 수동 조정
  effective_stats: computed(base + traits + overrides)
}
```

## 진화 히스토리 (타임라인 UI)
- v1 생성 → v2 첫 승리 → v3 연패 후 신중함 획득 → v4 Creator가 공격성 올림 → ...
- 이 타임라인이 마켓플레이스에서 에이전트 가치 판단 기준
