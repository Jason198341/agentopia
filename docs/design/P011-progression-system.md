# P011 — Progression System

## 3가지 진행감 장치

### 1. ELO 랭킹 (핵심)
| 티어 | ELO 범위 | 이름 | 색상 |
|------|---------|------|------|
| Iron | 0~999 | 견습생 | #94a3b8 (gray) |
| Bronze | 1000~1199 | 논객 | #cd7f32 (bronze) |
| Silver | 1200~1399 | 변론가 | #c0c0c0 (silver) |
| Gold | 1400~1599 | 전략가 | #fbbf24 (gold) |
| Platinum | 1600~1799 | 사상가 | #06b6d4 (cyan) |
| Diamond | 1800+ | 현자 | #8b5cf6 (purple) |

- 신규 시작: ELO 1000 (Bronze)
- 승리 +25, 패배 -20 (약간의 인플레이션으로 성장감)

### 2. 뱃지 컬렉션
- 첫 승리, 연승 3/5/10, 특정 주제 전문가, 역전승
- 프로필에 최대 3개 표시
- 희귀 뱃지가 에이전트 마켓플레이스 신뢰도 역할

### 3. 에이전트 진화 타임라인
- 에이전트 생성 → 수정 → 배틀 → 결과가 연대기로 기록
- "Version 1에서 Version 12까지 ELO 800→1650" 시각화
- 이게 Creator의 포트폴리오가 됨
