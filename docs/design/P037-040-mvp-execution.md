# P037~P040 — MVP Execution Plan

## P037: MVP Feature Scope

### 반드시 있어야 함 (Must-Have)
1. 에이전트 생성 (8 stats 슬라이더 + 프리셋)
2. 1:1 토론 배틀 (5턴, AI 심판)
3. ELO 랭킹 + 리더보드
4. 배틀 리플레이
5. 유저 인증 (Supabase Auth)
6. 프리미엄 구독 (Stripe)

### 없어도 됨 (Phase 2+)
- 관전자 투표 → Phase 2
- 마켓플레이스 → Phase 2
- 에이전트 토큰/투자 → Phase 3
- 팀전/토너먼트 → Phase 2
- 모바일 앱 → Phase 3
- B2B 기능 → Phase 2

## P039: 검증 지표 (출시 후 2주)

| 지표 | 합격선 | 불합격선 |
|------|--------|---------|
| 가입 수 | 100+ | <30 |
| D7 리텐션 | 40%+ | <15% |
| 배틀 완주율 | 80%+ | <50% |
| 에이전트 수정 횟수 (인당) | 3+ | <1 |
| 프리미엄 전환 | 5%+ | <1% |
| NPS | 30+ | <0 |

### 불합격 시 피벗 방향
1. **배틀이 재미없다** → 토론 → 퀴즈 대결로 포맷 변경
2. **에이전트 만들기 어렵다** → AI 자동 생성 + 원클릭 복제
3. **관전 수요가 더 크다** → Creator 아닌 Spectator 우선 피벗
