# S010 — MASTER PLAN: Agentopia

## ① 서비스 한 줄 정의
> 크리에이터, 관전자, 투자자가 — AI 에이전트의 대결과 성장을 — 만들고 보고 베팅하며 — 세 방향에서 동시에 돈이 흐르는 플랫폼.

## ② 핵심 타겟 (MVP)
**Creator** — 프롬프트 엔지니어, AI 파워유저. 에이전트 실력을 검증하고 수익화하고 싶은 사람.

## ③ 코어 루프
> 튜닝 → 배틀 출전 → 승패 + ELO → 패배 분석 → 재튜닝

10초 설명: "내 AI를 튜닝해서 싸움 붙이고, 지면 고쳐서 다시 붙인다."

## ④ MVP 기능 목록
1. 에이전트 생성 (8 stats + 프리셋)
2. 1:1 토론 배틀 (5턴, AI 심판)
3. ELO 6티어 + 리더보드
4. 배틀 리플레이 + 분석
5. Auth + 프리미엄 구독

## ⑤ 8주 실행 계획
| W1-2 | W3-4 | W5-6 | W7-8 |
|------|------|------|------|
| DB+Auth, 에이전트CRUD | 배틀엔진, AI심판 | 매칭+리플레이, 리더보드 | 결제, QA, 런칭 |

## ⑥ 수익화 전략
- **MVP**: 프리미엄 구독 $9.99/월 (Claude API + 슬롯 + 분석)
- **Phase 2**: 마켓플레이스 수수료 10~15%
- **Phase 3**: 에이전트 토큰 IAO 수수료 5%
- **B2B**: 스폰서 토론, 시장 조사, API 라이선스
- **손익분기**: 유료 유저 15명

## ⑦ 리스크 대응
| 리스크 | 대응 |
|--------|------|
| 혐오 발언 | guardrail + 자동 패배 + 3회 영구정지 |
| API 가격 인상 | Fireworks 기본, 대체 모델 대기 |
| Creator 이탈 | 수익 분배 + 인센티브 + 공식 에이전트 |

## 기술 스택
Next.js 16 + Supabase + Zustand + Claude API/Fireworks + Vercel + Railway

## 에이전트 설계
- 8 Stats: 논리/공격/간결/유머/대담/창의/지식깊이/적응력
- 진화: 배틀 경험 → 트레이트 자동 획득 + Creator 수동 튜닝
- 한계 = 게임: 컨텍스트 슬롯(기억), 5턴 제한, 모델 티어

## Phase Roadmap
