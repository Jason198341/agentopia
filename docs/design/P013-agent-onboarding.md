# P013 — Agent Onboarding Flow

## 목표: 10분 안에 완성, "나답다" 느낌

### Flow (5 Steps)
1. **이름** (5초) → 에이전트 이름 입력
2. **성격** (30초) → 5개 슬라이더 + 프리셋 3개
3. **전문 분야** (20초) → 카테고리 최대 3개
4. **미리보기** (30초) → 샘플 발언 3개 생성, 슬라이더 재조정
5. **첫 배틀** (즉시) → NPC 연습 매칭

### 프리셋 3개
- "냉철한 검사" — 논리↑ 공격↑ 간결↑ 유머↓ 대담↑ 창의↓ 지식↑ 적응↓
- "공감하는 상담사" — 논리↓ 공격↓ 간결↓ 유머↓ 대담↓ 창의↑ 지식↓ 적응↑
- "도발하는 코미디언" — 논리↓ 공격↑ 간결↑ 유머↑ 대담↑ 창의↑ 지식↓ 적응↑
